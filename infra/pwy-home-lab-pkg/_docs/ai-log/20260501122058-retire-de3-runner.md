# Retire de3-runner and pwy-home-lab-pkg

## Summary

Retired `~/git/pwy-home-lab-pkg` and `~/git/de3-runner` in favour of
`~/git/de3-generated-framework-repos/`. Both old repos are archived to
`~/git/_Archive/`. `de3-pwy-home-lab-pkg-repo` is now the primary working
location for running infrastructure — `./run -l` shows all 26 waves identically
to the old repo. All 13 generated repos now reference `de3-framework-pkg-repo`
as the source of `_framework-pkg` instead of `de3-runner`.

  copied from canonical old repo: bumped to v1.1.0, fixed `_modules_dir` from
  `default-pkg/_modules` (broken — `default-pkg` is pruned) to
  `_framework-pkg/_modules`

- **`de3-pwy-home-lab-pkg-repo/infra/pwy-home-lab-pkg/_config/_framework_settings/framework_packages.yaml`** —
  rewrote to list all 11 external packages (aws, azure, gcp, gui, image-maker,
  maas, mesh-central, mikrotik, proxmox, unifi) each from their respective
  `de3-*-pkg-repo` rather than de3-runner; `_framework-pkg` from
  `de3-framework-pkg-repo`

- **`de3-pwy-home-lab-pkg-repo/infra/pwy-home-lab-pkg/_setup/`** — copied
  `git-auth`, `git-auth-check.py`, `run`, `seed` from old repo

- **`de3-pwy-home-lab-pkg-repo/infra/pwy-home-lab-pkg/_docs/`** — migrated full
  ai-log, ai-plans, tech-debt history, README, framework-settings.md,
  git-auth.md, ssh-config.md

- **All 13 generated repos `framework_packages.yaml`** — `_framework-pkg` source

- **All 13 generated repos `framework_package_repositories.yaml`** — replaced

- **`~/git/_Archive/`** — `pwy-home-lab-pkg` and `de3-runner` moved here

- **`~/git/de3-ext-packages/de3-runner`** — removed entirely

## Root Cause

The generated repos were created when `pwy-home-lab-pkg` still used de3-runner
as the monorepo source for all packages. After this session all 13 repos now
reference their own dedicated generated repos, completing the migration to a
fully distributed repo structure.

## Notes

- `de3-runner` remains published on GitHub — it is not deleted, just locally
  archived. The `source_repo` entries in `framework_repos` now reference
  `de3-framework-pkg-repo`, which means future new-repo builds would use that
  as the rsync template (only `_framework-pkg` content, no per-package code).
- `demo-buckets-example-pkg` was omitted from `de3-pwy-home-lab-pkg-repo`'s
  `framework_packages.yaml` — it has no waves and no active infrastructure.
- `config/tmp/fw-repos-visualizer/` and `config/tmp/fw_repos_diagram_exporter/`
  files still reference de3-runner in stale output caches — these will be
  regenerated on the next visualizer run.
