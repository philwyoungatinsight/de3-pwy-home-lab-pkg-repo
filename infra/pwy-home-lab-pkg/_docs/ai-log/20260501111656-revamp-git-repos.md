# Revamp Git Repos — Fix Symlinks, Move Config

## Summary

Fixed all broken symlinks in the 13 generated framework repos under
`~/git/de3-generated-framework-repos/`, patched `fw-repo-mgr` to prevent the `.gitlab-ci.yml`
symlink from recurring, and placed the full `framework_repo_manager.yaml` in `de3-central-index-repo`
so it is now the canonical source for rebuilding the entire repo ecosystem.

## Changes

- **`~/git/de3-ext-packages/de3-runner/main`** (symlink, filesystem-only) — created; all
  generated repos' `_ext_packages/de3-runner/main` already pointed here but the target was
  missing. Now resolves to `~/git/de3-runner/main`.

- **`de3-{aws,azure,central-index,framework-pkg,gcp,gui,image-maker,maas,mesh-central,mikrotik,proxmox,pwy-home-lab-pkg,unifi}-pkg-repo/.gitlab-ci.yml`** — removed
  dangling symlink from all 13 generated repos and committed each. The symlink pointed to
  `infra/default-pkg/_framework/_git_root/.gitlab-ci.yml` which was always pruned.

- **`de3-runner: infra/_framework-pkg/_framework/_fw-repo-mgr/fw-repo-mgr`** — added 6-line
  fix after `prune_infra()`: detects if `.gitlab-ci.yml` is a dangling symlink and removes it,
  so future `fw-repo-mgr --build` runs don't recreate the broken symlink.

- **`de3-central-index-repo: infra/central-index-pkg/_config/_framework_settings/framework_repo_manager.yaml`** —
  copied from `pwy-home-lab-pkg`; 623 lines, all 13 ecosystem repos defined. Pushed to GitHub
  and GitLab.

- **`infra/pwy-home-lab-pkg/_docs/ai-plans/revamp-git-repos.md`** — archived (plan executed).

## Root Cause

`de3-runner` pre-dates the `de3-ext-packages` convention. Every other package lives under
`~/git/de3-ext-packages/<slug>/main`, but `de3-runner` was at `~/git/de3-runner/main` and
was never cloned into `de3-ext-packages`. `pkg-mgr` always creates `_ext_packages` symlinks
pointing into `~/git/de3-ext-packages/`, so the `_framework-pkg` chain was broken in every
generated repo. The fix is a single symlink — no code change required.

The `.gitlab-ci.yml` bug was a `fw-repo-mgr` oversight: it rsync'd the symlink from the
source repo but then pruned `default-pkg` (the symlink target), leaving it dangling.

## Notes

- The `de3-runner` symlink in `de3-ext-packages` is a filesystem-only fix — it is not tracked
  in any git repo. It will need to be recreated after a fresh machine setup or if
  `de3-ext-packages` is cleaned. A future step is to clone `de3-runner` directly into
  `de3-ext-packages` or retire it as `_framework-pkg` source entirely.
- `framework_repo_manager.yaml` was **copied** (not moved) — pwy-home-lab-pkg still has the
  original. The canonical location going forward is `de3-central-index-repo`.
- Steps 4 (retire de3-runner as `_framework-pkg` source) and 5 (retire pwy-home-lab-pkg) are
  deferred to a separate plan.
